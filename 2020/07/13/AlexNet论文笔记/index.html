介绍

AlexNet是一个大型的深层神经网络，能够对高分辨率(high resolution)的图像进行分类。AlexNet是2012年论文ImageNet Classification with Deep ConvolutionalNeural Networks中提出的一种卷积神经网络结构，并在12年的ImageNet分类赛上以大幅优势领先第二名，从而使得深度卷积神经网络CNN在图像分类上的应用掀起一波热潮，AlexNet有许多开创性举措，比如使用ReLU激活函数加速训练，采用了Dropout防止过拟合，使用双GPU并行训练加快训练速度。

网络结构

AlexNet由5个卷积层 + 3个全连接层构成。

非线性ReLU函数

使用ReLU激活函数比sigmoid或者tanh效果更好，因为sigmoid和tanh在深层神经网络的训练中容易造成梯度饱和，从而降低收敛的速度，使用ReLU函数训练速度更快。

Overlapping Pooling(重叠池化)

假设池化层的 kernel_size 为 z, 步长为 s。如果z = s, 则池化不会发生重合；如果z > s，则会出现重合。AlexNet使用 z = 3, s = 2的最大池化。训练模型发现使用重叠池化更难发生过拟合现象。

Local Response Normalization(局部响应归一化)

该方法未得到广泛应用，现在多用Batch Normalization进行归一化操作。

整体结构

输入224 x 224 x 3的三通道图像

  • 第一个卷积层：96个大小为11 x 11 x 3的卷积核，卷积步长 stride = 4，输出55 x 55 x 96大小的数据，然后经过最大池化，输出大小为27 x 27 x 96；
  • 第二个卷积层：256个大小为5 x 5 x 96的卷积核, same卷积，输出为27 x 27 x 256, 然后经过最大池化，输出为13 x 13 x 256;
  • 第三个卷积层：384个大小为3 x 3 x 256的卷积核，same卷积，输出为13 x 13 x 384;
  • 第四个卷积层：384个大小为3 x 3 x 384的卷积核，same卷积，输出为13 x 13 x 384;
  • 第五个卷积层：256个大小为3 x 3 x 384的卷积核，same卷积，输出为13 x 13 x 256, 然后经过最大池化，输出为6 x 6 x 256，将数据展开传到全连接层;
  • 三个全连接层：最后三个全连接层的神经元数量分别为4096、4096、1000.

降低过拟合

数据增强

  1. 图像平移和水平反射。通过从256×256幅图像中提取随机224×224块图像(及其水平反射)，其实就是随机剪裁（crop），并在这些提取的图像上训练我们的网络。这将我们的训练集的大小增加了2048倍。

  2. 改变了训练图像中RGB通道的强度。在整个imagenet训练集中对RGB像素值集执行PCA操作。

Dropout

采用keep_prob = 0.5的Dropout正则化。这种技术减少神经元之间的相互适应，因为神经元不能依赖其他神经元的存在。Dropout使迭代次数加倍。
